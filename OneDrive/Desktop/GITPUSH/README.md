# ayush14

A premium comprehensive package containing TypeScript, React, Redux, GraphQL, and Zod boilerplate, utilities, assignments, and sample architectures for modern web development.

Developed and maintained by **AYUSH14**.

---

## 🚀 Overview

This package aggregates a rich ecosystem of assignments, boilerplate modules, and reference implementations. Whether you are learning full-stack development, reviewing advanced patterns, or integrating specific workflows (like Zod validation, Redux slices, or GraphQL resolvers), this package serves as a central reference.

---

## 📂 Included Components & Projects

Here is a catalog of the modules included in this workspace:

### 1. 📚 Full-Stack Projects
* **Book Store Management (`Book_Store_Mannagment`)**: Complete Apollo GraphQL integration with TypeDefs, Resolvers, MongoDB models on the backend, and standard React frontend components.
* **Pagination (`Pagination`)**: Complete server-side paginated product lists using Redux Toolkit query (RTK) on the frontend and Express/MongoDB on the backend.
* **Real-Time Chat (`RealTimeChat`)**: Node.js/Socket.io backend with a React real-time messaging frontend.

### 2. 🌀 GraphQL Ecosystem
* **Customer Support (`GraphQL/CustomerSupport`)**: Ticket queries, server setup, and support workflow schemas.
* **Nested GraphQL (`GraphQL/NestedGraphQl`)**: Complex relational data resolver modeling.
* **Student Info System (`GraphQL/Student_Information_system`)**: Relational student/course registration resolvers.
* **GraphQL Frontend (`GraphQl_Frontent`)**: Apollo client configuration and React component binding.

### 3. 🔐 Forms & Validations
* **User Registration ZOD (`UserRegistrationZOD`)**: Premium form handling and reactive feedback using [Zod](https://zod.dev/) schemas.
* **Seller Registration (`SellerRegistration`)**: Seller signup validations.
* **Controlled Form (`ControlFormUseState`)**: Advanced stateful controlled components using React `useState`.
* **Uncontrolled Form (`UnControlled`)**: Reference logic for ref-based uncontrolled inputs.

### 4. 🧠 State Management & Hooks
* **Mini E-commerce Redux (`MiniEcommerce(Redux)`)**: Slices for cart, products, and user actions using `@reduxjs/toolkit`.
* **Context API (`Context API Assignment`)**: Prop drilling avoidance reference using React Context and Providers.
* **UseReducer (`UseReducer`)**: Complex local state transitions using reducer patterns.
* **Performance Optimization (`PerformaceOptimization`)**: Dynamic optimization components featuring `React.memo`, `useMemo`, and `useCallback`.

### 5. 🎨 Interactive Utilities & Games
* **Tic Tac Toe (`Tic_Tac_To.ts`)**: Interactive React game logics.
* **Typing Speed (`typingspeed.ts`)**: Real-time typing speed and WPM tracker utility.
* **Food Rating (`food_rating_app.tsx`)**: Reactive star rating and feedback component.
* **Show & Hide Password (`Show_and_hidepassword.ts`)**: Client-side password field toggle utility.
